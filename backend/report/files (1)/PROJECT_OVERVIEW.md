# 监管违规穿透式查询系统 - 项目总览

## 项目简介

本系统是一个基于社区检测和知识图谱技术的监管违规事件智能分析平台。系统能够自动分析监管违规事件，识别涉及的责任方、违规行为和监管机构，并基于预构建的社区结构和知识图谱生成深度分析报告。

## 核心功能

1. **智能实体提取**：使用DeepSeek API从事件描述中自动提取关键实体
2. **社区匹配**：将实体与三种视角的社区结构进行智能匹配
3. **链路构建**：基于社区层级关系构建完整的事件传导路径
4. **网络可视化**：生成直观的网络图展示事件结构
5. **深度分析**：结合社区报告和AI能力生成专业分析报告

## 系统特色

### 1. 多视角分析
从责任方、违规行为、监管机构三个维度全方位分析事件，确保分析的全面性。

### 2. 知识图谱支持
基于真实的资本市场法规构建的知识图谱，包含：
- 599个实体节点
- 1427条关系边
- 覆盖11个监管机构社区
- 覆盖15个责任方社区
- 覆盖13个违规行为社区

### 3. 智能匹配算法
- 精确匹配：完全匹配实体名称
- 包含匹配：子串匹配
- 模糊匹配：关键词相似度匹配
确保即使描述不够精确也能找到相关社区。

### 4. 可视化呈现
使用NetworkX和Matplotlib生成高质量的网络图：
- 不同颜色区分不同视角
- 箭头表示关系方向
- 线条粗细表示关联强度

### 5. AI深度分析
利用DeepSeek大模型的能力，结合社区报告生成：
- 责任主体分析
- 违规行为认定
- 法律依据引用
- 监管措施建议
- 合规风险提示

## 使用流程

### 准备阶段
1. 安装依赖：`pip install -r requirements.txt --break-system-packages`
2. 配置API密钥
3. 验证数据：`python3 test_system.py`

### 运行阶段
1. 运行系统：`python3 regulatory_query_system.py`
2. 输入事件描述
3. 等待处理（通常30秒-2分钟）
4. 查看输出结果

### 输出结果
- `event_network.png` - 网络图
- `community_reports.txt` - 社区报告
- `final_analysis.txt` - 综合分析
- `analysis_summary.xlsx` - Excel表格

## 性能指标

- **实体提取准确率**：取决于DeepSeek API性能
- **社区匹配召回率**：通过多级匹配策略提高
- **处理时间**：通常30-120秒
- **支持事件长度**：不限（建议100-500字）
