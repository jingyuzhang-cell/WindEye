# 输出文件格式说明

## Step1 输出

目录：`backend/report_outputs/gnn_leiden_results`

每个视角默认产出 5 个文件：

- `*_1_节点聚类结果.csv`
- `*_2_关系聚类结果.csv`
- `*_3_社区统计摘要.csv`
- `*_4_跨社区关系分析.csv`
- `*_visualization_data.json`

三种视角总计 15 个核心文件。

## Step2 输出

目录：`backend/report_outputs/community_reports`

文件：

- `责任方社区报告.xlsx`
- `违规行为社区报告.xlsx`
- `监管机构社区报告.xlsx`

## Step3 输出

目录：`backend/report_outputs/build_hierarchy_links_output`

文件：

- `community_hierarchy_gnn.xlsx`
- `community_hierarchy_gnn.png`
- `community_quality_report.xlsx`

## 样例与历史产物

旧版本样例、历史结果和原型代码已归档到：

- `backend/new_report/code/legacy`
- `backend/new_report/docs/archive`
