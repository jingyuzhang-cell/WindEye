# 社区报告生成

## 对应代码

- [step2_community_reports.py](/D:/Code/WindEye/backend/new_report/code/run_pipeline/step2_community_reports.py:1)
- [generate_community_reports.py](/D:/Code/WindEye/backend/new_report/code/generate_community_reports.py:1)

## 输入

Step2 依赖两类输入：

- Step1 的聚类结果 CSV/JSON
- 政策法规原文 TXT

## 输出

默认生成三份 Excel：

- `监管机构社区报告.xlsx`
- `违规行为社区报告.xlsx`
- `责任方社区报告.xlsx`

输出目录是 `backend/report_outputs/community_reports`。

## 报告字段

Excel 中保留以下核心字段：

- `id`
- `community`
- `level`
- `title`
- `summary`
- `key_words`
- `findings`
- `rank`
- `rating_explanation`
- `full_content_json`

## LLM 策略

- 已配置 `DEEPSEEK_API_KEY` 或 `LLM_API_KEY` 时，调用 DeepSeek 生成结构化报告
- 未配置时，回退为基于图谱和统计结果的基础报告
