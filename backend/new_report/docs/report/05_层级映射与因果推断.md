## 分层定义

- `L0`：责任方社区
- `L1`：违规行为社区
- `L2`：监管机构社区

## 连接评分

当前主流程采用两类信号进行加权：

- `Section/Law` 重叠度：0.5
- 语义相似度：0.5

默认阈值：

- 责任方 → 违规行为：`>= 0.30`
- 违规行为 → 监管机构：`>= 0.30`
- 责任方 → 监管机构：`>= 0.25`
