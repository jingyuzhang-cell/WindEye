# 审查规则库

## 一、内容真实性规则（10 项）

| 规则 ID | 规则名称 | 审查要点 | 法规依据 |
|---------|---------|---------|---------|
| R01 | 虚假信息 | 不得含有虚假或引人误解的内容 | 第十条第（一）项 |
| R02 | 数据真实性 | 引用数据必须真实、准确、已核实 | 第十条第（二）项 |
| R03 | 保本承诺 | 不得明示或暗示保本 | 第十条第（三）项 |
| R04 | 收益承诺 | 不得承诺收益或限定损失 | 第十条第（三）项 |
| R05 | 业绩预测 | 不得预测未来业绩 | 第十条第（三）项 |
| R06 | 模拟业绩 | 不得利用模拟业绩误导投资者 | 第十条第（三）项 |
| R07 | 保险夸大 | 不得夸大保险责任或收益 | 第十条第（四）项 |
| R08 | 简单类比 | 不得将保险与存款简单类比 | 第十条第（四）项 |
| R09 | 备案误导 | 不得利用备案程序误导消费者 | 第十条第（五）项 |
| R10 | 诱导消费 | 不得片面宣传诱导消费 | 第十条第（六）项 |

## 六、敏感词库（80+ 违规词）

### 通用敏感词
- 保本、保息、保本保息、零风险、稳赚不赔
- 高收益、低风险、低门槛、秒到账、无成本
- 限时抢购、抢购、秒杀、抄底、暴富、财富自由
- 市场第一、遥遥领先、最强、最佳、顶级

### 基金行业敏感词
- 坐享其成、躺赚、抄底逃顶
- 专家推荐、受益者证明
- 目标价 XX 元、强烈推荐

### 证券行业敏感词
- 目标价 XX 元、强烈推荐
- 内幕消息、确定性判断

### 保险行业敏感词
- 存钱送保障、分红 XX%
- 不用健康告知

### 银行行业敏感词
- 预期年化收益、存款送礼品
- 保本理财
