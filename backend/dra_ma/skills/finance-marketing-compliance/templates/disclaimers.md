# 金融风险提示模板库

---

## 一、通用风险提示

### 简短版
```
⚠️ 投资有风险，入市需谨慎。
```

### 标准版
```
⚠️ 风险提示：
- 过往业绩不代表未来表现
- 不构成投资建议
- 请根据自身风险承受能力选择产品
- 市场有风险，投资需谨慎
```

### 完整版
```
⚠️ 重要风险提示：

1. 市场风险：投资品价格可能波动，存在本金损失风险
2. 流动性风险：部分产品存在封闭期，无法随时赎回
3. 信用风险：发行人可能无法按期兑付
4. 政策风险：监管政策变化可能影响投资收益

本文内容仅供参考，不构成任何投资建议。
投资有风险，决策需谨慎。
```

## 二、基金产品专用提示

### 业绩展示提示
```
📈 业绩说明：
- 过往业绩不代表未来表现
- 基金净值会波动，可能跌破面值
- 历史最高收益不代表能保证
- 请查看完整历史业绩（含亏损年份）

数据来源：基金定期报告，截至 XXXX 年 XX 月 XX 日
```

### 风险等级提示
```
⚠️ 风险等级说明：
R1（低风险）- R5（高风险）
您的风险承受能力：C1-C5
请确保产品风险等级 ≤ 您的风险承受能力
```

### 费用提示
```
💰 费用说明：
- 申购费：XX%（购买时收取）
- 管理费：XX%/年（每日计提）
- 托管费：XX%/年（每日计提）
- 赎回费：XX%（持有<7 天收取 1.5%）
```

### 标准风险提示语（法规要求）
```
基金有风险，投资需谨慎。
过往业绩不代表未来表现。
基金管理人承诺以诚实信用、勤勉尽责的原则管理和运用基金资产，
但不保证基金一定盈利，也不保证最低收益。
```

## 三、保险产品专用提示

### 犹豫期提示
```
📋 犹豫期提示：
本产品设有 XX 天犹豫期（自签收合同次日零时起算）。
犹豫期内退保，无损失退还已交保费。
犹豫期后退保，仅退还现金价值，可能有损失。
```

### 健康告知提示
```
⚠️ 健康告知重要提示：
- 请如实告知健康状况
- 不如实告知可能导致拒赔
- "熬过两年就能赔"是误导
- 有既往症请先咨询核保
```

### 分红险提示
```
📊 分红险特别说明：
- 分红是不保证的
- 历史分红不代表未来
- 分红水平取决于公司实际经营
- 最坏情况下可能无分红
```

### 万能险提示
```
📊 万能险特别说明：
- 最低保证利率之上的收益是不确定的
- 结算利率每月公布，历史利率不代表未来
- 前期会扣除初始费用
```

### 投连险提示
```
📊 投连险特别说明：
- 投资账户收益浮动，可能产生亏损
- 不承诺保本保收益
- 投资风险完全由客户承担
```

## 四、银行理财专用提示

### 净值化提示
```
📊 净值型理财说明：
- 本产品为净值型，非保本
- 净值会波动，可能跌破 1
- 预期收益率≠实际收益率
- 资管新规后无保本理财
```

### 业绩比较基准提示
```
📊 业绩比较基准说明：
- 业绩比较基准 XX%，不代表未来表现
- 不构成收益承诺
- 实际收益以产品净值为准
```

### 流动性提示
```
🕐 流动性说明：
- 封闭期：XX 天（期间不可赎回）
- 开放期：每 X 天开放一次
- 赎回确认：T+X 日到账
- 提前赎回可能有损失
```

### 标准风险提示语
```
理财非存款，产品有风险，投资需谨慎。
本产品不承诺保本保收益。
```

## 八、适当性匹配提示

### 风险测评提示
```
📋 风险测评重要提示：
- 请如实填写风险测评问卷
- 测评结果有效期：XX 个月
- 情况变化后请重新测评
- 不得代客户填写测评
```

### 匹配原则
```
✅ 适当性匹配原则：
C1（保守型）→ R1（低风险）
C2（稳健型）→ R1-R2
C3（平衡型）→ R1-R3
C4（成长型）→ R1-R4
C5（进取型）→ R1-R5
```

```
"您好，我是 XX 保险公司代理人 XXX，工号 XXX。

```

### 产品介绍
```
"向您推荐的是 XX 保险产品，保险期间 XX 年，保费 XX 元，
保额 XX 元。保险责任包括...（详见条款）"
```

### 风险提示确认
```
"请您确认已知悉以下事项：
1. 这是保险产品，不是银行存款
2. 退保可能有损失
3. 分红是不保证的（如适用）
4. 健康告知需如实填写
您是否理解并确认？"

客户："我已理解并确认。"
```

### 双录结束
```
"感谢您的配合，录音录像结束。"
```
