# 金融营销合规助手 v2.0.0

**定位**：基于《金融产品网络营销管理办法》及 15+ 金融监管法规，提供金融产品营销内容合规审查和方案生成。

**合并说明**：由 `content-compliance`（v1.0.0）和 `finance-marketing-compliance`（v1.0.0）合并而成，保留双方优势。

---

## 🎯 核心功能

### 功能 1：文章合规审查

**触发**："检查这篇文章合规吗"、"审查营销文章"

**输入**：
- 文章全文（文本/链接/文件）
- 金融产品类型（存款/贷款/证券/保险/基金等）

**输出**：
- 合规状态（✅ 合规 / ⚠️ 需修改 / ❌ 违规）
- 违规条款（具体条款号 + 内容）
- 修改建议
- 风险等级（低/中/高）

**审查维度**：
1. 内容真实性（第十条）
2. 关键信息完整性（第八条）
3. 禁止行为（第十条 8 项）
4. 资质要求（第十八条）
5. 适当性管理（第二十条）

### 功能 2：营销方案合规审查

**触发**："营销方案合规审查"、"检查营销方案"

**输入**：
- 营销方案全文
- 营销渠道（公众号/直播/短视频/网站等）
- 金融产品类型

**输出**：
- 合规状态
- 违规条款
- 修改建议
- 风险等级

**审查维度**：
1. 渠道合规性（第五条、第十六条）
2. 内容合规性（第七条、第八条、第十条）
3. 行为规范（第十一条至第十九条）
4. 合作规范（第二十条至第二十七条）

### 功能 3：分行业合规审查

**触发**："审查基金营销"、"检查保险话术"

**支持行业**：
- 基金行业（专属词库 + 审查规则）
- 证券行业（专属词库 + 审查规则）
- 保险行业（专属词库 + 审查规则）
- 银行行业（专属词库 + 审查规则）

**输出**：
- 合规状态
- 违规条款
- 修改建议
- 风险等级

### 功能 4：合规营销方案生成

**触发**："生成合规营销方案"、"制定营销方案"

**输入**：
- 金融产品类型
- 目标人群
- 营销渠道
- 核心卖点

**输出**：
- 完整营销方案
- 合规提示
- 风险点标注
- 修改建议

## ⚠️ 常见错误

**错误 1：忽略法规时效性**
```
问题：
• 使用过期的法规版本
• 未标注法规实施日期

解决：
✓ 始终使用最新法规版本
✓ 标注法规实施日期（2026-09-30）
✓ 定期更新知识库
```

**错误 2：违规判断过于绝对**
```
问题：
• 直接判定"违规"，不说明原因
• 不考虑具体场景和上下文

解决：
✓ 标注具体违规条款
✓ 说明违规原因
✓ 提供修改建议
```

**错误 3：忽略金融产品类型差异**
```
问题：
• 对所有金融产品使用同一套规则
• 不区分存款、贷款、证券、保险等

解决：
✓ 根据产品类型选择对应规则
✓ 标注适用的法规条款
✓ 考虑产品特殊性
```

## 🧪 使用示例

**输入 1：文章合规审查**
```
审查这篇文章：
《XX 银行高收益存款产品推荐》
年化收益 5.8%，保本保息，限时抢购！

产品类型：存款
```

**输出 1**：
```
🔍 合规审查结果
━━━━━━━━━━━━━━━━━━━━
合规状态：❌ 违规
风险等级：高

违规条款：
1. 第十条第（三）项：明示或暗示保本、承诺收益
   - "保本保息"违反规定

2. 第十条第（七）项：使用诱导性用语
   - "高收益"、"限时抢购"属于诱导性用语

修改建议：
1. 删除"保本保息"表述
2. 删除"高收益"、"限时抢购"等诱导性用语
3. 补充风险提示
━━━━━━━━━━━━━━━━━━━━
```

**输入 2：营销方案合规审查**
```
审查这个营销方案：
渠道：微信公众号
产品：基金
方案：通过公众号推文推荐 XX 基金，使用"低风险高收益"表述，附购买链接
```

**输出 2**：
```
🔍 合规审查结果
━━━━━━━━━━━━━━━━━━━━
合规状态：❌ 违规
风险等级：高

违规条款：
1. 第十条第（七）项：使用诱导性用语
   - "低风险高收益"违反规定

2. 第五条：跳转至非自营平台
   - 附购买链接需跳转至金融机构自营平台

修改建议：
1. 删除"低风险高收益"表述
2. 购买链接需跳转至金融机构自营平台
3. 补充产品风险提示
━━━━━━━━━━━━━━━━━━━━
```

**输入 3：合规营销方案生成**
```
生成营销方案：
产品：存款
目标人群：30-50 岁工薪族
渠道：微信公众号
核心卖点：年化收益 3.5%，期限灵活
```

**输出 3**：
```
📋 合规营销方案
━━━━━━━━━━━━━━━━━━━━
产品：XX 银行定期存款
目标人群：30-50 岁工薪族
渠道：微信公众号

【标题】
XX 银行定期存款产品说明

【正文】
XX 银行定期存款，年化收益率 3.5%，期限灵活可选。

【风险提示】
存款产品受存款保险条例保护，具体收益以实际为准。

【合规提示】
✅ 未使用诱导性用语
✅ 未承诺收益
✅ 补充风险提示
━━━━━━━━━━━━━━━━━━━━
```
